# Task 2: Dart Basic Functions

Proyek ini dibuat untuk memenuhi tugas pemrograman Dart yang mengimplementasikan fungsi-fungsi dasar berdasarkan tiga topik utama. Setiap topik dirancang menggunakan empat tipe fungsi yang berbeda untuk mendemonstrasikan pemahaman mengenai parameter dan nilai kembalian (*return value*).

---

## 📂 Struktur Repositori

Proyek ini dibagi menjadi tiga file utama sesuai dengan topik instruksi:

1.  **`topik_menyapa.dart`**
    Berisi fungsi untuk menyapa pengguna dengan berbagai variasi format teks.
2.  **`topik_kalkulator_luas.dart`**
    Berisi logika matematika untuk menghitung luas persegi menggunakan berbagai pendekatan fungsi.
3.  **`topik_kalkulator_diskon.dart`**
    Berisi fungsi kalkulasi untuk menghitung potongan harga dan total pembayaran belanja.

---

## 🛠️ Detail Teknis

Setiap topik memiliki 4 jenis fungsi berbeda:

* **Tanpa Parameter & Tanpa Return Value (`void`)**: Fungsi menjalankan aksi statis di dalam blok kode.
* **Dengan Parameter & Tanpa Return Value (`void`)**: Fungsi menerima input untuk diproses dan ditampilkan langsung.
* **Tanpa Parameter & Dengan Return Value**: Fungsi mengembalikan nilai data tertentu tanpa memerlukan input eksternal.
* **Dengan Parameter & Dengan Return Value**: Fungsi menerima input, memprosesnya, dan mengembalikan hasil akhir ke pemanggil.

---

## 🚀 Cara Menjalankan

Pastikan Anda memiliki Dart SDK terinstal di lingkungan Anda. Jalankan file menggunakan perintah berikut di terminal:

## Menjalankan Topik Menyapa

```bash
dart run topik_menyapa.dart
```

## Menjalankan Topik Kalkulator Luas

```bash
dart run topik_kalkulator_luas.dart
```

## Menjalankan Topik Kalkulator Diskon
```bash
dart run topik_kalkulator_diskon.dart
```
